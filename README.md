<p>Projeto gerado através do curso <b>Angular: Avançando no design de componentes com acessibilidade</b> da plataforma Alura. </p>

<p>Neste projeto foi gerado um modal de forma dinâmica e acessível.</p>

- Diretivas
- Services
- Modules
- Forms reativo
